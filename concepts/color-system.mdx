## 柱状图与热力图颜色：方向与强度

Nightwatch 中柱状图和热力图的颜色直接来自实时计算——

- **绿色系**：该行权价的 netGex 为正（稳定倾向）
- **红色系**：该行权价的 netGex 为负（波动放大倾向）
- **颜色深浅 = 相对强度**：越深代表该节点的 GEX 越集中，越浅代表 GEX 越分散。最深的那个行权价就是 King Strike。

这不是人为标注的"看涨/看跌"，而是期权持仓数据驱动的客观映射。

## 行权价标记颜色：节点角色

除了绿/红色阶之外，行权价列表中的**文字颜色**和**边框**用来标识该行权价在当前结构中的特殊角色：

| 视觉标记 | 含义 | 出现条件 |
|----------|------|---------|
| **金色文字 + ◆ 菱形** + 左侧金色边框 | King Strike——GEX 绝对值最大的节点 | 所有到期分组 |
| **紫色文字** + 左侧紫色边框 | Gatekeeper——GEX 绝对值第二大的节点 | 所有到期分组 |
| **青色文字** + 右侧青色边框 | Active Call——当日到期合约中 Call 成交最活跃 | 仅 0DTE 和 1DTE |
| **琥珀色文字** + 右侧琥珀色边框 | Active Put——当日到期合约中 Put 成交最活跃 | 仅 0DTE 和 1DTE |
| **白色高亮行** | 当前现货价格所在行 | 始终显示 |
| 普通灰色文字 | 无特殊角色 | 其他行权价 |

<Tip>
一个行权价可以同时拥有左侧边框（结构角色）和右侧边框（成交活跃度）。例如，King 行权价同时也是 Active Call 时，你会看到左侧金色边框 + 右侧青色边框同时出现。
</Tip>

## 如何快速读取整体环境

- 绿色为主 → 正 GEX 主导，当日震荡偏稳，大波动概率低
- 红色为主 → 负 GEX 主导，当日动量偏强，突破后加速风险高
- 上绿下红或上红下绿的对称分布 → 市场在某一行权价两侧存在明显 GEX 分歧，该价位是结构边界

## 热力图颜色的额外含义

- **0DTE 热力图**：颜色跟踪资金流向（近 5 分钟内成交量驱动的 GEX 变化），反映"钱正在往哪里流"
- **Weekly/Monthly 热力图**：颜色跟踪持仓结构，反映"钱已经在哪里布局"

## 一个容易犯的错误

<Warning>
深暗色/高透明度的格子 = 那个时间段没有数据（例如盘前），**不是** GEX 为零。

极浅中性色的格子 = netGex 真正接近零。

两者完全不同，不要混淆。
</Warning>
