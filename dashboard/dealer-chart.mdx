## 这个功能给你什么

Dealer GEX 图表是 Nightwatch 正在测试的新视图——它把做市商敞口数据直接叠加在 K 线图上，用发光线段（**PULSE**）标记出做市商定位最集中的几个关键价位。

K 线走势和做市商压力分布在同一张图上，不需要在面板之间来回切换。

<Note>
  此功能仍在测试中，仅对部分用户开放。如果你在 Dashboard 的视图切换中看不到 Chart 选项，说明该功能尚未对你开启。
</Note>

## 与现有 GEX 视图的区别

Dashboard 中已有的 GEX 视图展示的是**市场整体的 Gamma 结构**——它基于全市场参与者的持仓分布，告诉你哪些行权价聚集了最多的 Gamma 敞口。

Dealer GEX 的视角不同。它尝试从市场活动中**分离出做市商一方的定位**，捕捉做市商在日内是如何不断调整自身风险敞口的。两者观察的维度不同，得到的结果也不同——它们是互补关系，不是替代关系。

<Warning>
  Dealer GEX 的建模方法仍在测试和优化中。当前精度不代表最终表现，请结合现有 GEX 视图综合判断。
</Warning>

## 什么是 PULSE

PULSE 是 K 线图上叠加的**发光横向线段**。每一条 PULSE 标记一个做市商敞口集中的关键价位。图表默认显示当前影响力最大的 5 个 PULSE。

<Warning>
  PULSE 的视觉样式、数量和渲染方式在测试期间可能发生变化。
</Warning>

### 两种颜色

<CardGroup cols={2}>
  <Card title="靛蓝色 PULSE — 正 Gamma" icon="shield">
    做市商的对冲行为会**抑制波动**——价格在这些区域趋于平稳，运动放缓。

    像减速带——价格靠近时容易被"黏住"，形成震荡或反弹。
  </Card>
  <Card title="金色 PULSE — 负 Gamma" icon="bolt">
    做市商的对冲行为会**放大波动**——价格在这些区域趋于剧烈，运动加速。

    像加速器——价格进入时容易被推动，出现快速拉升或暴跌。
  </Card>
</CardGroup>

### 亮度 = 强度

PULSE 的亮度代表该价位的影响力大小：

| 亮度 | 含义 |
|------|------|
| 暗淡 | 影响力较小 |
| 明亮 | 影响力较大 |
| 带辉光 | 当前最强的 PULSE，对价格有最大的吸引力 |

## 怎么看图表

<Steps>
  <Step title="找到最亮的 PULSE">
    最亮的那条（带辉光效果）就是当前最重要的价位。价格在收盘时经常被"钉"在这个位置附近。
  </Step>
  <Step title="看上下分布">
    当前价格**上方**最强的 PULSE = 短期阻力区；**下方**最强的 PULSE = 短期支撑区。这两条 PULSE 之间就是价格当天最可能活动的区间。
  </Step>
  <Step title="注意空白区域">
    如果两条 PULSE 之间有一大段没有标记的空白，说明那个区域缺少做市商力量。价格一旦进入，容易快速穿越。
  </Step>
  <Step title="跟踪变化">
    PULSE 会随盘中交易实时更新。某个方向突然出现更亮的 PULSE，说明做市商定位正在向那个方向倾斜。
  </Step>
</Steps>

## 重要提示

<Warning>
  Dealer GEX 图表和 PULSE 目前处于测试阶段。以下局限性在正式版中可能会改善，但目前请务必了解：
</Warning>

- **PULSE 反映的是做市商当前的定位状态，不是价格预测。** 突发事件会瞬间改变局面。
- **先看 K 线结构，再看 PULSE 确认。** 它是辅助判断工具，不是交易信号。
- **价格不会精准在 PULSE 位置反弹。** 小幅偏差是正常的，看的是区域而非精确点位。
- **PULSE 的颜色和亮度会随盘中交易变化。** 被反复触碰的 PULSE 可能被改写——盯**实时变化**比盯最初状态更重要。
- **PULSE 每天从零开始。** 每个交易日都是一张独立的地图，开盘前 15 分钟 PULSE 可能很稀疏，这是正常现象。
- **测试期间模型仍在迭代。** PULSE 的精度和覆盖范围会随着后续更新持续优化。

## 术语速查

| 术语 | 含义 |
|------|------|
| **GEX** | Gamma Exposure——衡量特定价位上 Gamma 敞口的集中程度 |
| **Dealer GEX** | 从做市商视角观察的 Gamma 敞口分布 |
| **PULSE** | K 线图上的发光线段标记，显示 Dealer GEX 影响力最大的价位 |
| **正 Gamma（靛蓝色）** | 做市商定位倾向于抑制波动的区域 |
| **负 Gamma（金色）** | 做市商定位倾向于放大波动的区域 |

</Info>
