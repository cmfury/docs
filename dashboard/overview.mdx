## 这个页面给你什么

GEX Dashboard 是 Nightwatch 的核心。它把所有上市期权合约的持仓数据实时聚合、计算、映射成你能直接读懂的结构图——你不需要看期权链、不需要手动算 Gamma，只需要打开页面，关键节点和市场结构一目了然。

## 顶部导航栏

| 元素 | 说明 |
|------|------|
| **标的下拉** | 切换观察标的（SPY、QQQ 等） |
| **0DTE / 1DTE / WEEKLY** | 三个到期分组，每个分组对应不同的持仓时间框架 |
| **LIVE 指示灯** | 绿色脉冲 = 实时更新；灰色 = 连接中断（会自动重连） |
| **MKT OPEN / MKT CLOSED** | 美东时间市场状态 |
| **GEX / VOL 模块切换** | 切换到 Volatility 面板 |

## 工作流建议

1. **开盘前**：先看 Weekly GEX 判断本周结构，再切到 1DTE 预判今天的 Gamma 格局
2. **开盘后**：切到 0DTE，找 King Strike 和 Gamma Flip
3. **日内**：实时观察热力图，感知结构节点是否在漂移；用 1DTE 辅助判断隔夜持仓方向
